# Data Analysis Checklist

## Before calculation

- Are all units explicit and compatible?
- Are concentration, fraction, percentage, and derived variables clearly distinguished?
- Are the required formulas grounded in the current experiment materials?
- Are assumptions stated where the handout is silent?

## During calculation

- Keep intermediate quantities visible.
- Use consistent significant figures.
- Preserve a trail from raw measurements to final parameters.
- Separate transformed variables from measured variables.

## During fitting

- State the fit form.
- State the extracted parameters.
- Check whether the fitted result has physical meaning.
- Flag suspicious deviations or outliers without deleting them silently.

## During interpretation

- Separate random error from systematic error.
- Do not overstate conclusions beyond what the data support.
- If trends are reliable but absolute values are shifted, state both parts clearly.
