# Raw Data Verification

## Purpose

Raw-data transcription from images is a high-risk step because a single copied number can distort the entire downstream analysis.

## Required behavior

- Preserve original labels, units, and trial order.
- Mark uncertain readings explicitly instead of guessing silently.
- Export `??????.csv` before any derived calculation.
- Present the table to the user for verification.
- Do not continue until the user explicitly confirms the transcription.
