# Postlab Rules

Do not generate `reports/postlab_report.html` until:

- raw data are confirmed
- analysis is complete
- analysis report is approved
- final figures exist in `figures/`

The postlab must use:

- `source/` report requirements
- `analysis/raw-data/verified_raw_data.csv`
- `analysis/processed/processed_data.csv`
- `analysis/processed/summary_data.csv`
- `analysis/processed/calculation_chain.md`
- final figure files from `figures/`

Do not use placeholders for missing figures.
