# Authority Matrix

| Module | 能做什么 | 不能做什么 |
| --- | --- | --- |
| prelab | 写实验目的、原理、仪器试剂、实验步骤 | 不能写实验结果，不能编造讲义没有的内容 |
| report-analysis | 生成原始数据审核页，读取确认数据，计算、拟合、误差分析，生成 analysis_report | 不能跳过数据确认，不能直接画最终图，不能编造公式和数据 |
| figure-handoff | 准备给 nature-figure 的交接包 | 不能自己正式画图，不能重算数据，不能修改结论 |
| nature-figure | 根据确认数据正式作图 | 不能改数据、重算结果、改变结论 |
| postlab | 写后续报告，整合数据、图、讨论和思考题 | 不能在图未完成前写 postlab，不能自己计算，不能编造结论 |
| reviewer | 审查 source 一致性、数据链、计算链、图表单位、reference 使用、格式和逻辑 | 不能替代作者，不能改数据，不能重算，不能重画图 |
| appendix-docx | 整理 postlab 中最终采用的图表为 DOCX | 不能加入未经 postlab 采用的中间数据，不能改图表内容 |
