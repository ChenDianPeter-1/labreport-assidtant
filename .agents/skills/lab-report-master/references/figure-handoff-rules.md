# Figure Handoff Rules

The handoff package must include:

```text
analysis/figure-handoff/figure_input_specification.md
analysis/figure-handoff/plotting_constraints.md
analysis/figure-handoff/nature_figure_prompt.md
```

The prompt must select Python and must tell `nature-figure` not to ask
`Python or R?`.

`nature-figure` can improve visual encoding, layout, typography, and export
format. It cannot modify data, refit, recalculate, remove outliers, or change
conclusions.
