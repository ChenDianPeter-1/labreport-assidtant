# Appendix DOCX Rules

Build:

```text
reports/appendix_figures_tables.docx
```

Extract only final adopted figures and tables from:

```text
reports/postlab_report.html
```

Default format:

- three-line tables
- table title above table
- figure caption with every figure
- table caption with every table
- order follows postlab
- no discussion prose

Render-check the DOCX and repair layout issues before delivery.
