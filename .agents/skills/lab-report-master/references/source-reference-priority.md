# Source And Reference Priority

## source/

`source/` is the hard authority for the current experiment:

- handout
- teacher requirements
- raw records
- raw data
- report template
- experiment procedure
- data-processing requirement
- unit requirement
- plotting requirement

Use `source/` to decide facts, formulas, units, steps, data treatment, report
structure, and required figures.

## reference/

`reference/` is supporting material:

- papers
- good examples
- supplementary notes
- reference reports
- materials for better wording or plotting ideas

Use `reference/` only to improve explanation, wording, background, and visual
planning. It must not override `source/`.

## Conflict Rule

When `source/` and `reference/` conflict:

1. Follow `source/`.
2. Record the conflict in the reviewer section.
3. Ask the user if the conflict affects formulas, units, data processing, or
   conclusions.
