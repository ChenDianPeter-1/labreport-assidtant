# HTML Output Rules

Allowed HTML files:

```text
reports/prelab_report.html
analysis/raw-data/raw_data_review.html
analysis/report/analysis_report.html
reports/postlab_report.html
```

Rules:

- Chinese interface
- simple local HTML
- printable layout
- no server required
- no complex dashboard
- clean tables
- clear source and generated-file notes
- collapsible reviewer details
- collapsible long calculation details

Use `<details>` for detail-heavy sections.
