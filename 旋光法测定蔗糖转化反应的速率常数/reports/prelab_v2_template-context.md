# prelab_v2 模板上下文

本上下文按新版 `lab-report-writing` 的预习报告流程建立，用于约束 `reports/prelab_v2.html` 的结构和内容来源。

## 1. 实验目的

- 模板要求：简明写出本实验要达到的目的；说明通过哪些测量、计算或图像关系得到核心结论。
- 讲义来源：`Ⅰ、目的要求`，包括测定蔗糖转化反应速率常数和半衰期、了解浓度与旋光度关系、掌握旋光仪使用。
- reference 用途：仅辅助确认旋光法跟踪蔗糖转化反应的教学背景。
- 避免越权：不写当前实验的实测速率常数、半衰期或结论。

## 2. 实验原理

- 模板要求：简明说明主要原理；写出后续数据处理直接相关的公式、符号含义和适用条件。
- 讲义来源：`Ⅲ、实验原理`，包括蔗糖水解反应、酸催化准一级处理、旋光度与浓度正比、用 \(\ln(\alpha_t-\alpha_\infty)\) 对 \(t\) 作图求 \(k\)。
- reference 用途：仅作为背景资料，不替代讲义公式。
- 避免越权：不引入外部文献值、理论速率常数、不确定度或拟合结果。
- 修订记录：根据用户反馈，将本节压缩为核心反应、准一级近似、旋光度-浓度关系和作图求 \(k\) 四个必要逻辑点，删除展开式背景说明。

## 3. 实验装置、设备及试剂

- 模板要求：列出主要装置、设备和试剂；必要时提供装置示意图。
- 讲义来源：`Ⅱ、仪器与试剂`，包括 WZZ-2B 自动旋光仪、样品管、秒表、恒温槽、量筒、锥形瓶、蔗糖溶液和盐酸。
- reference 用途：不需要使用外部资料补充设备清单。
- 避免越权：不添加讲义未列出的仪器、规格或试剂浓度。

## 4. 实验步骤（预习准备工作）

- 模板要求：不照抄教材；用流程图或分步说明呈现实验流程；标出关键点、难点、待观察现象和待测数据；涉及配方或预计算时给出相应结果。
- 讲义来源：`Ⅳ、实验步骤`、`Ⅴ、实验数据记录`、`ⅤI、数据处理`。
- reference 用途：辅助确认操作逻辑，不增加讲义外步骤。
- 避免越权：不进行正式数据计算、拟合、误差分析或实验结论撰写。
