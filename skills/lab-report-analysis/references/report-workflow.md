# Report Workflow

This skill covers the data-analysis stage only.

## Scope

Use this workflow after the experiment has been performed and the user wants to turn raw measurements into analyzable results.

Do not absorb the pre-lab report into this skill.

## Default stage order

1. inspect current experiment materials
2. transcribe raw-data images into `??????.csv`
3. pause for user verification
4. propose the calculation chain
5. pause for user verification
6. execute data processing and fitting
7. pause for user verification
8. package report-ready outputs

## Pause rule

Each major stage should end with a short review gate:

- what was completed
- what should be checked
- what will happen next after confirmation
